# StudentOS

StudentOS is an AI-powered student operating system designed for:

- Nursery Schools
- Primary Schools
- Secondary Schools
- Universities

## Features

- AI Tutor
- Homework Scanner
- OCR Text Extraction
- Exam Practice
- Notes System
- Study Planner
- Timetable
- Flashcards
- GPA Calculator
- Voice Learning
- Offline Learning
- Scholarship Alerts
- Career Guidance

## Supported Exams

- JAMB
- WAEC
- NECO
- GCE
- SAT
- IELTS
- TOEFL
